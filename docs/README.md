# Student Reference Materials

This folder contains optional reference material for the four active projects.
Most students should start from the root `README.md` and then read the README in
the project folder they are working on.

`lecture-assets/` contains optional figures and notes used to explain P01/P02
signal chains and interpretation.  They are supporting material, not required
