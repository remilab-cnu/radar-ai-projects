# P05: Lightweight Radar Waveform Classification Example

P05 is a compact Python equivalent of MATLAB radar waveform-classification
examples.  It generates simple complex-baseband waveform observations, converts
them to STFT log-magnitude images, and trains a small CNN classifier.

## MATLAB reference

- [Radar and Communications Waveform Classification Using Deep Learning](https://www.mathworks.com/help/radar/ug/radar-and-communications-waveform-classification-using-deep-learning.html)
- [LPI Radar Waveform Classification Using Time-Frequency CNN](https://www.mathworks.com/help/radar/ug/LPI-radar-waveform-classification-using-time-frequency-CNN.html)

The Python example keeps the same teaching idea—waveform family classification
from time-frequency structure—but uses a small synthetic dataset and a tiny CNN
so it can run on a CPU.

## Task

Classify four baseband observation types from a 64x64 STFT image:

`rect`, `lfm`, `barker`, `noise_only`

This is waveform classification, not range-Doppler target detection.  The input
is one received waveform observation; the output is the waveform family label.

## Python equivalent

```text
waveform family + random pulse settings
    -> frequency offset + AWGN
    -> STFT log-magnitude image
    -> handcrafted baseline or tiny CNN classifier
```

## Architecture

- **Model**: TinyWaveformCNN
- **Input**: `(B, 1, 64, 64)` STFT log-magnitude image
- **Output**: `(B, 4)` class logits
- **Loss**: CrossEntropyLoss with light label smoothing
- **Baseline**: handcrafted waveform descriptors + logistic regression/SVM

## Commands

```bash
# Fast CPU check
python train.py --generate --smoke

# Default lightweight run
python train.py --generate --epochs 10

# Eval only
python train.py --eval_only --checkpoint artifacts/best_model.pt

# Classical feature baseline
python evaluate_feature_baseline.py --data_dir data --model logreg
python evaluate_feature_baseline.py --data_dir data --model rbf_svm --max_train 2000

# SNR-bin evaluation
python evaluate_snr_sweep.py --checkpoint artifacts/best_model.pt

# Example STFT images
python plot_examples.py
```

If a checkpoint was created with `--smoke`, add `--base_ch 8` to eval commands
that instantiate the neural model.

## Data

Generated by `generate_data.py` using `shared/waveform_library.py`.

| Split | Filename | Default size |
|---|---|---:|
| Train | `data/waveform_train.h5` | 4000 |
| Val | `data/waveform_val.h5` | 800 |
| Test | `data/waveform_test.h5` | 800 |

HDF5 schema includes `x (N,1,64,64)`, `y (N,)`, `features (N,F)`, `snr_db`, pulse
metadata, STFT settings, class names, and MATLAB reference URLs.

## Metrics

| Metric | Description |
|---|---|
| accuracy | Overall class accuracy |
| f1_macro | Macro-averaged F1 across waveform classes |
| confusion_matrix | Which waveform families are confused |
| per-SNR accuracy | Accuracy grouped by SNR bin |

## Lightweight exploration knobs

- `--snr_lo`, `--snr_hi`: noise difficulty
- `--epochs`, `--lr`, `--base_ch`: learning and model capacity
- `evaluate_snr_sweep.py --bins`: SNR-bin interpretation
- `evaluate_feature_baseline.py --model`: classical baseline family
